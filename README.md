Description:
End-to-end automated test suite for Temenos T24 using Playwright, following the Page Object Model (POM) design pattern.
This project focuses on modern automation practices for web banking systems, ensuring stability and parallel execution across browsers.

Key Features:

Built with Playwright for fast and reliable browser automation.

Page Object Model structure for modularity and easier maintenance.

Supports parallel testing and cross-browser execution (Chromium, Firefox, WebKit).

Includes custom utilities for environment setup, login flow, and dynamic wait handling.

Generates HTML/JSON reports with detailed test results.

Tech Stack: Playwright, JavaScript/TypeScript, Node.js, POM, Allure/HTML Reporter
